# Line Follower 
Цель проекта: Реализовать алгоритм управления роботом, следующего по линии в симуляторе MuJoCo с использованием компьютерного зрения (OpenCV).

Задачи:

Настроить симуляцию робота в MuJoCo.

Реализовать алгоритм обработки изображения с камеры робота для детектирования линии.

Технологии: Python, MuJoCo, OpenCV, NumPy.

Структура проекта: 
line_follower/
├── models/
│   └── line_follower.xml        # Модель робота и окружения в MuJoCo
├── src/
│   ├── line_follower.py         # Основной класс робота и логика управления
│   ├── main.py                  # Точка входа в программу
│   └── simulation.py            # Управление симуляцией и визуализацией
├── .gitignore
├── Dockerfile                   
├── README.md
├── config.py                    # Конфигурационные параметры
└── requirements.txt             # Зависимости Python

Модульная архитектура: 

simulation.py — работа с MuJoCo, рендеринг.

line_follower.py — логика робота, обработка изображения, управление.

main.py — точка входа.

Запуск:  
1. Установить зависимости:  
   pip install -r requirements.txt
   Основные зависимости:
   mujoco
   opencv-python
   numpy
2. Запустить детектор
    python src/main.py
